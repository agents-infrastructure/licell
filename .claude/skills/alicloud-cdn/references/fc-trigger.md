# Function Compute Trigger

Edge function triggers with Alibaba Cloud Function Compute.

## addFCTrigger

**Signature:** `addFCTrigger(request: AddFCTriggerRequest)`

Adds a Function Compute trigger..

**Parameters:** See `AddFCTriggerRequest` model.

## updateFCTrigger

**Signature:** `updateFCTrigger(request: UpdateFCTriggerRequest)`

Updates a specified Function Compute trigger..

**Parameters:** See `UpdateFCTriggerRequest` model.

## deleteFCTrigger

**Signature:** `deleteFCTrigger(request: DeleteFCTriggerRequest)`

Deletes a specified Function Compute trigger..

**Parameters:** See `DeleteFCTriggerRequest` model.

## describeFCTrigger

**Signature:** `describeFCTrigger(request: DescribeFCTriggerRequest)`

Queries a specified Function Compute trigger..

**Parameters:** See `DescribeFCTriggerRequest` model.

## listFCTrigger

**Signature:** `listFCTrigger(request: ListFCTriggerRequest)`

> You can call this operation up to 100 times per second per account..

**Parameters:** See `ListFCTriggerRequest` model.
