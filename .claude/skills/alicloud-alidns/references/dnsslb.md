# DNS Load Balancing (DNSSLB)

DNS-based load balancing for sub-domains with weighted round-robin.

## setDNSSLBStatus

**Signature:** `setDNSSLBStatus(request: SetDNSSLBStatusRequest)`

Enables or disables weighted round-robin based on the specified parameters..

**Parameters:** See `SetDNSSLBStatusRequest` model.

## describeDNSSLBSubDomains

**Signature:** `describeDNSSLBSubDomains(request: DescribeDNSSLBSubDomainsRequest)`

Queries the subdomains for which weighted round-robin is enabled based on the specified parameters..

**Parameters:** See `DescribeDNSSLBSubDomainsRequest` model.

## updateDNSSLBWeight

**Signature:** `updateDNSSLBWeight(request: UpdateDNSSLBWeightRequest)`

Modifies the weight of a Domain Name System (DNS) record based on the specified parameters..

**Parameters:** See `UpdateDNSSLBWeightRequest` model.
